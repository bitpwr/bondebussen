# Bondebussen

Web page based on vue.js, node.js and express showing current time table for
public transports in Stockholm.
In particular, busses from Bondevägen.

## Configuration

The followiong environment variables must be set
```
export BB_REALTIME_KEY=<sl-realtime-key>
export BB_SEARCH_KEY=<sl-stationsearch-key>
export BB_SERVER_PORT=<port>
```
